# 🩺 MaternalPredict — Advanced Maternal & Fetal Health Prediction Platform

## 🌸 Introduction

**Maternal Predict** is an AI-powered platform designed to revolutionize maternal and fetal healthcare through **predictive analytics**.
By leveraging machine learning models and real-time data visualization, MedPredict empowers both **healthcare professionals** and **expectant mothers** to identify potential pregnancy risks early — promoting timely interventions and healthier outcomes.

---

## 🚀 Key Features

### 🤰 Pregnancy Risk Prediction

Predict potential pregnancy complications using **advanced algorithms** trained on key maternal parameters such as:

* Age
* Blood pressure
* Blood sugar levels
* Body Mass Index (BMI)
* Lifestyle and medical history

By analyzing these inputs, MedPredict offers **personalized insights** and **early warning signals** to aid in preventive healthcare planning.

---

### 👶 Fetal Health Prediction

Maternal Predict integrates **Cardiotocogram (CTG)** analysis — a reliable, cost-effective method to assess fetal well-being.
Our AI-driven model classifies fetal states into **normal**, **suspect**, or **pathological**, helping healthcare professionals take **timely and informed action** to reduce maternal and neonatal mortality risks.

---

### 📊 Interactive Dashboard

A **comprehensive and intuitive dashboard** gives users a complete overview of predictions and health metrics.

* Visualize patient data trends and risk probabilities
* Manage, compare, and monitor multiple cases
* Gain actionable insights through clean data visualization

Designed for simplicity, speed, and accessibility — no advanced technical skills required.

## ⚙️ How to Run Locally

Follow these simple steps to set up MedPredict on your local machine:

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/maternalpredict.git
cd maternalpredict
```

### 2️⃣ Install Dependencies

Ensure you have Python ≥ 3.8 installed, then run:

```bash
pip install -r requirements.txt
```

### 3️⃣ Launch the Streamlit App

```bash
python - m streamlit run main.py
```

Once launched, open the local URL provided in your terminal to explore **MaternalPredict** on your browser.
